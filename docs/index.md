# はじめに

本教材は主にロボット競技参加者およびロボットの研究活動をROS2を使ってされたい方向けに執筆しています。
本教材では、ROBOTIS社の[Turtlebot3](https://e-shop.robotis.co.jp/list.php?c_id=93)をgazebo上でシミュレーションし、簡易なプランナーを動かします。
環境構築等が容易、確実に行えるように、Dockerを用いて本教材は作られています。
